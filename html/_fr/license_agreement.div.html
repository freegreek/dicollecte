Les dictionnaires de Dicollecte sont publiés sous licences open source. Les ajouts, les modifications et les propositions faits pour l’amélioration des dictionnaires seront soumis aux licences de ceux-ci.

Les fichiers sont actuellement publiés sous les licences suivantes :

  • Dictionnaires orthographiques : LGPL, GPL, MPL
  • Dictionnaire des synonymes : LGPL
  • Lexique : LGPL, GPL, MPL

Je comprends les principes de l’open source et je les accepte. J’accepte que les dictionnaires puissent éventuellement être diffusés sous d’autres licences open source reconnues par l’Open Source Initiative.

Je n’ajouterai pas dans le système des données qui ne sont pas libres de droit.
