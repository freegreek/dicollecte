About

The Dicollecte project goal is to improve dictionaries for OpenOffice.org, Firefox and Thunderbird.

This gate collects propositions from users. Subscribing is mantarory.

This gate allows to :

  • explore the dictionaries,
  • propose new words,
  • correct mistakes,
  • comment propositions,
  • modify the thesaurus,
  • validate users propositions (controllers only).

Each proposition will be examined by controllers and possibly corrected. Validated propositions will be added to the next dictionary version.
