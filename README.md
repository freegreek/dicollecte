dicollecte
==========

### A collaborative tool to work on Hunspell dictionaries:

 * add, remove, edit entries of the dictionaries
 * add, remove, edit entries in synsets
 * edit the affixation rules (administrators only)

### A tool similar to a “bug tracker”:

 * suggestions made by users can be committed in the dictionary, after tagging and discussions
 * a log stores modifications
 * inflexions generation on the fly to track mistakes

### Other features:

 * multi-project management
 * multilingual user interface
 * projects customization
 * editable pages
 * sub-dictionaries management
 * preset tags for entries
 * 4 levels of privileges: administrators, editors, controllers, simple users
