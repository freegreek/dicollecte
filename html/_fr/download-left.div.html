Installation des dictionnaires

LibreOffice / OpenOffice.org

Lire le tutoriel d’installation des dictionnaires sur le forum de la communauté francophone.

Firefox & Thunderbird

Ce mode d’installation est doréravant déconseillé et n’est présenté qu’à titre informatif.

1. Pour l’installation manuelle, allez dans :

[répertoire d’installation]\dictionaries\

2. Effacez les anciens fichiers s’il y en a. [optionnel]

3. Décompressez un ou plusieurs dictionnaires de votre choix dans ce répertoire.

4. Redémarrez le navigateur.


Note : L’installation manuelle a le même effet que si vous utilisez les extensions, mais contrairement aux extensions, ces dictionnaires ne sont pas automatiquement mis à jour.

Vous pouvez choisir quel dictionnaire utiliser avec un clic droit dans une zone de texte → Langues.

Chrome / Chromium

Les dictionnaires ne peuvent être utilisés directement. Il est nécessaire de les convertir en fichiers binaires.

1. Téléchargez l’archive contenant le programme de conversion et décompressez-la.

2. Téléchargez le dictionnaire de votre choix et décompressez l’archive.

3. La commande pour compiler les dictionnaires est :

convert_dic fr-mondico

Il faut omettre l’extension des fichiers pour désigner le dictionnaire choisi. La compilation génère un fichier fr-mondico.bdic.

4. Allez dans :

[répertoire d’installation de Chrome]\dictionaries\

5. Remplacez le fichier .bdic par celui que vous avez obtenu.

Opera

Opera permet de télécharger des dictionnaires, grâce à un clic droit dans un champ de texte. → Dictionnaires → Ajouter/supprimer des dictionaires… Les dictionnaires sont installés dans votre profil.

Note : le dictionnaire français proposé actuellement est une ancienne version de 2009.

Linux

Contrairement à Windows, où chaque logiciel gère le dictionnaire à sa façon, les distributions Linux intègrent Hunspell et les dictionnaires comme des dépendances externes aux logiciels, mais elles le font aussi chacune à leur manière.

En général, tous les dictionnaires sont stockés dans un répertoire commun. Vous trouverez ce répertoire en cherchant les fichiers .aff ou .dic. Par exemple, sous Ubuntu :

/usr/share/hunspell

Les répertoires dictionaries des logiciels ne contiennent que des liens symboliques vers ce dossier.

D’une manière générale, les dictionnaires français sont disponibles dans le paquet hunspell-fr.
